# 🌟 Interactive Particle Hello World

웹 기술을 활용한 화려하고 인터랙티브한 "HELLO WORLD" 파티클 애니메이션 프로젝트입니다.

## 🚀 주요 기능

- **실시간 파티클 텍스트**: 수천 개의 파티클이 모여 "HELLO WORLD" 글자를 형성합니다.
- **마우스 인터랙션 (밀어내기)**: 마우스 커서를 글자 위로 가져가면 파티클들이 커서를 피해 부드럽게 흩어집니다.
- **클릭 이벤트 (폭발 효과)**: 마우스를 클릭하고 있으면 반발력이 강화되어 파티클들이 더 멀리 폭발하듯 흩어집니다.
- **실시간 정보 표시**: 화면 하단에 현재 날짜, 시간 및 접속 위치(IP 기반) 정보를 실시간으로 표시합니다.
- **고급스러운 디자인**: Glassmorphism 스타일과 다크 모드 테마가 적용되었습니다.

## 🛠 사용된 기술

- **프론트엔드**: HTML5, CSS3 (Vanilla CSS), JavaScript
- **그래픽**: HTML5 Canvas API
- **폰트**: Google Fonts (Inter, Segoe UI)

## 📁 프로젝트 구조 및 버전

Git을 통해 체계적으로 버전 관리가 되어 있습니다.

- **`ver1.0`**: 초기 HELLO WORLD 페이지 구성
- **`ver2.0`**: 인터랙티브 파티클 효과 구현
- **`ver3.0`**: 날짜, 시간, 위치 정보 인터페이스 추가 (최신 버전)

## 💻 실행 방법

별도의 서버 설치 없이 웹 브라우저만 있으면 바로 실행 가능합니다.

1. 이 저장소를 클론(Clone)하거나 다운로드합니다.
2. `index.html` 파일을 크롬(Chrome)이나 엣지(Edge) 등 현대적인 브라우저로 엽니다.
3. 화면을 클릭하거나 마우스를 움직여 파티클과 상호작용해 보세요!
